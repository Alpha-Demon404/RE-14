# Instatools

<br>A tools for instagram <br>

## How to install?
```
$ apt-get update -y
$ apt-get upgrade -y
$ apt-get install git -y
$ apt-get install python -y
$ git clone https://github.com/asmin-dev/Instatools
$ cd Instatools
```

**Module required**


* 1). Cython 
    * ```pip install cython```
* 2). requests 
    * ```pip install requests```
* 3). BeautifulSoup
    * ```pip install bs4```

```
$ python install.py 
$ ./run
```
